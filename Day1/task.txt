1. Write a blog on Difference between HTTP1.1 vs HTTP2

HTTP1.1:
•	It transfer all the request and response in the text message form.
•	It blocks all the request until it doesn’t get its resources.
•	It compresses data by itself.
HTTP 2:
•	It works on binary framing that covert all text message to binary format.
•	It allow multiplexing so multiple request can be sent at a time.
•	It uses HPACK for data compression.
